# Выпуклый анализ и глубокое обучение

В репозитории находятся выполненные лабораторные работы по курсу «Выпуклый анализ и глубокое обучение».

## Структура

- `lw/lw1/Lab_1_solution.ipynb` — лабораторная работа 1: классификация CIFAR-10 с помощью полносвязных нейронных сетей.
- `lw/lw2/Lab_2_solution.ipynb` — лабораторная работа 2: реализация MLP на NumPy и сравнение с готовыми библиотеками.
- `lw/lw3/Lab_3_solution.ipynb` — лабораторная работа 3: сравнение ResNet18 и ResNet18 с Self-Attention.

## Краткие результаты

### Лабораторная работа 1

Задача: классификация CIFAR-10 без использования CNN.

Итоговая `test accuracy`: около `0.5682`.

### Лабораторная работа 2

Задачи:

- многоклассовая классификация Iris;
- регрессия Diabetes.

Реализован собственный класс `MyMLP` на NumPy.

Лучший результат MyMLP на Iris: `accuracy` около `0.9667`.

Лучший результат MyMLP на Diabetes: `R2` около `0.4281`.

Также выполнено сравнение со `sklearn`, `Keras/TensorFlow` и `PyTorch`.

### Лабораторная работа 3

Задача: сравнение стандартной ResNet18 и ResNet18 с добавленным Self-Attention на CIFAR-10.

Baseline ResNet18 `test accuracy`: около `0.7675`.

ResNet18 + Self-Attention `test accuracy`: около `0.7650`.
